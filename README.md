# Braap

Motocross performance tracking platform powered by Haiden Deegan.

Track your rides, chase the leaderboard, and compete in Danger Boy Challenges.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- Framer Motion
- Supabase (auth, database)
- Stripe (subscriptions)
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
